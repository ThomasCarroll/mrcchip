MRC CSC ChIP seq QC pipeline

The MRC chipseq pipeline provides basic processing, analysis and quality control applicable to ChIP-seq and other targeted enrichment sequencing technologies. The full description of these steps can be seen below. Currently this is designed for use on the Imperial ax3 cluster using pbs pro v 12.2:

To get a working copy of this project please use the version installed on the ax3. (Contact thomas.carroll@imperial.ac.uk).

If you would like the review the source code download from this link. 

$ cd your_mydir/dir_name
$ wget https://github.com/ThomasCarroll/mrcchip/archive/master.zip ./
$ unzip master.zip
